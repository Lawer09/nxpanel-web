## Bug 修复

- 修复投放管理线上登录跨域预检失败的问题：AdsConsole 请求线上不再从 `/ads-api` 改写为 `https://console.adsmakeup.com/api`，统一保留同源 `/ads-api/*` 路径，避免浏览器直连跨域接口触发重复 `Access-Control-Allow-Origin` 拦截（`src/requestErrorConfig.ts`, `docs/issue/ads-console.md`）
- 修复广告变现同步节点弹窗点击下一页不刷新数据的问题：同步节点列表改为受控分页，并在翻页时携带 `page/pageSize` 重新请求接口（`src/pages/ad/ad-revenue/components/SyncServersModal.tsx`, `src/services/ad/api.ts`）
- 修复版本检查接口首屏重复请求：为 `guest/version/latest` 增加进行中请求共享与 3 秒短缓存，避免浏览器刷新时页头、版本弹窗和运行时检查各自重复拉取（`src/services/version/api.ts`）
- 修复代理流量账号管理弹窗中“流量分配”误用广告账户的问题：分配弹窗改为加载代理流量账号列表，默认当前行且允许切换来源流量账号；目标用户固定为当前点击行，提交分配接口时不再因切换流量账号而改动目标用户信息（`src/pages/traffic-platform/dashboard/components/AccountManageModal.tsx`）
- 修复 DNS 管理执行解析弹窗中“唯一解析/唯一绑定”默认值错误的问题：表单初始值与打开弹窗预设统一改为默认否，和接口文档中的 `unique=false` 保持一致（`src/pages/dns/index.tsx`）
