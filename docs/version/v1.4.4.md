# v1.4.4

## 优化功能

- 优化项目管理导入弹窗为双模式入口：同一弹窗支持 `上传 CSV` 与 `粘贴文本` 两种导入源，粘贴模式可直接识别无表头项目块并复用现有 `batch-save` 提交流程（src/pages/project-table/components/ProjectBatchImportModal.tsx, src/pages/project-table/pasteImport.ts）。
- 优化项目管理导入入口文案：工具栏按钮改为“导入数据”，导入弹窗步骤标题中的首步改为“上传数据”，其余 CSV 格式提示保持不变（src/pages/project-table/index.tsx, src/pages/project-table/components/ProjectBatchImportModal.tsx）。
- 优化项目报表项目代号列的限流状态展示：`isLimited=false` 时不再显示“正常”tag，仅保留“限流”和“未知”两种提示，减少标签噪音（src/pages/report/project/index.tsx）。
- 优化项目报表平台/投放状态展示方式：将 `appPlatform`、`adStatus` 从项目代号右侧 tag 收口为仅前端展示的可选维度列，默认隐藏、依赖 `projectCode` 才可勾选，且对应筛选项改为跟随漏斗显示（src/pages/report/project/index.tsx, src/components/report/UniversalReportTable.tsx）。

## Bug 修复

- 修复项目管理无表头粘贴导入只能依赖 CSV 表头映射的问题，新增按 `projectCode` 切分项目块并自动识别负责人、包名、域名/Admob/Yandex/Firebase 等字段的解析能力（src/pages/project-table/components/ProjectBatchImportModal.tsx, src/pages/project-table/pasteImport.ts, docs/api/project_api.md）。
- 修复项目管理粘贴导入对低置信度内容缺少保护的问题，缺少项目代号、字段冲突或存在未识别内容的记录现在会在预览中标记为“跳过/异常”并默认排除，避免错误入库（src/pages/project-table/components/ProjectBatchImportModal.tsx, src/pages/project-table/pasteImport.ts, docs/issue/project_table.md）。
- 修复项目管理粘贴导入将 Facebook token / hash 长串误判为项目代号的问题，项目块边界现在仅接受 `Demumu` 或 `大写字母 + 数字 + 可选大写字母` 的项目代号格式（src/pages/project-table/pasteImport.ts, docs/issue/project_table.md）。
- 修复项目管理粘贴导入异常提示过于模糊的问题，预览表现在会直接显示具体异常说明，并将冲突字段翻译为中文导入字段名，同时展示“已有值 / 新值”，便于快速定位问题（src/pages/project-table/components/ProjectBatchImportModal.tsx, src/pages/project-table/pasteImport.ts, docs/issue/project_table.md）。
- 优化项目管理粘贴导入预览表布局：`状态` 与 `异常说明` 固定列宽度收窄为紧凑展示，内容改为单行省略并通过 tooltip 查看完整信息，减少左侧固定列对横向空间的占用（src/pages/project-table/components/ProjectBatchImportModal.tsx）。
- 修复项目管理粘贴导入对商店页、隐私协议、服务条款 URL 识别不稳定的问题，现已改为严格命中 `play.google.com/store/apps/details`、`privacy_policy.html`、`terms_of_use.html` 才归类到对应字段，并保留 `privacy.html / terms.html` 兼容，避免普通链接误判（src/pages/project-table/pasteImport.ts, docs/issue/project_table.md）。
- 修复项目管理粘贴导入中 Yandex 账号与广告位混淆的问题，`@yandex.com` 和 `未分配账号` 识别为 `Yandex账号`，带 `Yandex` 前缀且符合 `R-M-xxx-x` 格式的文本识别为 `Yandex广告ID`（src/pages/project-table/pasteImport.ts, docs/issue/project_table.md）。
- 修复项目管理粘贴导入将 `Yandex未接入` 误判为未识别内容的问题，该文案现按 `Yandex广告ID` 字段收口，不再进入异常说明（src/pages/project-table/pasteImport.ts, docs/issue/project_table.md）。
