# v1.4.3

## 优化功能

- 优化项目管理表格页批量维护能力：列表支持多选项目，并在选中后提供“修改投放状态”入口，调用批量更新接口统一修改 `adStatus`，单次限制最多 500 个项目（src/pages/project-table/index.tsx, src/services/project/api.ts, src/services/project/types.ts）。
- 项目报表增加 `appPlatform` 筛选，并在项目编码列展示应用平台 tag，便于按 `iOS / Android` 维度快速识别与筛选项目（src/pages/report/project/index.tsx, src/pages/project/constants.ts, src/services/report/typings.d.ts）。
- 优化项目管理表格页字段维护能力：新增 `appPlatform`（应用平台）字段展示与编辑，并支持列表多选后批量更新应用平台，保持与现有项目接口和详情资源绑定联动刷新（src/pages/project-table/index.tsx, src/pages/project-table/fields.ts, src/services/project/api.ts, src/services/project/types.ts）。
- 优化项目管理表格页应用信息分组：`appPlatform` 调整到“应用信息”中展示，并在新建、编辑、批量修改时统一使用 `IOS` / `Android` 下拉选择输入（src/pages/project-table/fields.ts, src/pages/project-table/index.tsx, src/pages/project/constants.ts）。
